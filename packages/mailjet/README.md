## Versioning

Follow [Tekartik git versioning system](https://github.com/tekartik/common.dart/blob/main/doc/tekartik_versioning.md)

## Setup

```yaml
dependencies:
  tekartik_mailjet:
    git:
      url: https://github.com/alextekartik/http_apis.dart
      path: packages/mailjet
      ref: dart2_3
    version: '>=0.1.0'
```
